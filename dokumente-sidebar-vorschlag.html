Baumdarstellung

Unterhalb der Hauptnavigation erscheint eine aufklappbare Ordnerstruktur. Ordner lassen sich per Klick auf den Chevron aufklappen, um Unterordner zu sehen. Ein Klick auf den Ordnernamen navigiert direkt dorthin.

Badges mit Dateianzahl

Jeder Ordner zeigt rechts ein Badge mit der Anzahl enthaltener Dateien. Leere Ordner erhalten ein dezentes "0"-Badge in grau, gefuellte Ordner ein oranges Badge. So erkennt man sofort, wo Inhalte liegen.

Aufklappbare Ebenen

Die Baumstruktur unterstuetzt beliebig tiefe Verschachtelung. Vertikale Fuehrungslinien helfen, die Zugehoerigkeit der Unterordner visuell nachzuvollziehen. Der aktuell aktive Ordner wird orange hervorgehoben.

Synchronisierte Navigation

Baum und Hauptbereich sind synchronisiert: Klickt man einen Ordner im Baum, oeffnet sich der Inhalt rechts. Navigiert man per Breadcrumb, passt sich der Baum an. So hat der Benutzer jederzeit den vollen Ueberblick.

Voller/Leerer Ordner-Icon

Ordner mit Inhalt zeigen ein gefuelltes Icon (bi-folder-fill), leere Ordner ein leeres Icon (bi-folder). Zusammen mit dem Badge ergibt sich ein klares visuelles Signal auf einen Blick.

Lazy Loading

Unterordner werden erst beim Aufklappen vom Server geladen (Lazy Loading). So bleibt die Sidebar performant, auch bei grossen Ordnerstrukturen. Ein kurzer Spinner zeigt den Ladevorgang an.
